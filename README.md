# Malozna Task Bot (робоча версія)

Це мій особистий Telegram-бот, який я поступово розширюю з SMM-бота до повноцінного мінімалістичного органайзера: задачі, нотатки, пошук, експорт. Робиться під мій стиль роботи — без зайвого шуму і складних меню.

## Що вже зроблено
- оформлено команди бота через BotFather
- підготовлена структура під задачі та нотатки
- визначений стиль відповідей: коротко, спокійно, мінімалізм

## Що буде додано
- `/new` — створити нову задачу через діалог
- `/tasks` — переглянути активні задачі
- `/done` — закрити задачу за ID
- `/note` — швидка нотатка в один меседж
- `/notes` — список останніх нотаток
- `/search` — пошук по задачах і нотатках
- `/export` — вивантажити все у .txt
- `/ai_note` — оформлення задачі через AI
- `/ai_summary` — підсумок дня

## Стек
Python · Aiogram 3 · SQLite · dotenv  
(далі буде OpenAI інтеграція)

## Ідея
Створюється як особистий робочий інструмент. Мікс задачника, нотатника і простого AI-помічника.
