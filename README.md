# smm-sender-bot

Простий Telegram-бот для ручних SMM-розсилок: зберігає підписників та відправляє їм повідомлення за командою адміністратора.

## Стек

- Python 3.12
- Aiogram 3
- python-dotenv
- JSON-файл як проста база підписників

## Можливості

- /start — додає користувача до списку підписників
- /help — коротка довідка
- /send <текст> — розсилка повідомлення всім підписникам (лише для ADMIN_ID)

## Запуск (локально)

```bash
pip install -r requirements.txt
python src/bot.py
```
